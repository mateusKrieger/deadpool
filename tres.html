Conceitos servidor adicionais

O que é o Apache?

O Apache é um servidor de código aberto e nome oficial é Apache HTTP Server, mantido pela Apache Software Foundation, e alimenta cerca de 46% de todos os sites hospedados na internet.

O que é o Tomcat?

O software Tomcat, desenvolvido pela Fundação Apache, permite a execução de aplicações para web.

O que é uma LAN?

Uma rede local (LAN) conecta dispositivos que estão fisicamente próximos uns dos outros usando conectores como roteadores e switches. Ela permite que os dispositivos troquem dados e se comuniquem com segurança em pequena escala.

O que é MAN?

Uma rede de área metropolitana (MAN) é uma rede de computadores que conecta computadores dentro de uma área metropolitana, que pode ser uma única grande cidade, várias cidades e vilas, ou qualquer área com vários edifícios. Uma MAN é maior que uma rede local (LAN) mas é menor que uma rede de longa distância (WAN).

O que é uma WAN?

O que é WAN? Rede de longa distância (WAN) é uma rede que abrange uma área geográfica de grande escala. Seu modem envia e recebe informações de e para a Internet pela porta WAN.

O que é uma PAN?

Uma rede de área pessoal (PAN) conecta dispositivos eletrônicos dentro da área imediata de um usuário. O tamanho de uma PAN varia de alguns centímetros a alguns metros. Um dos exemplos mais comuns do mundo real de uma PAN é a conexão entre um fone de ouvido Bluetooth e um smartphone.
